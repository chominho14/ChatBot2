### ChatBot 토이프로젝트

---

##### 프로젝트 계획 이유

> 많은 기업들의 챗봇을 직접 구현하고싶은 마음에 계획하게 됐다.

---

#### 개발환경

- Language : Typescript
- Framework : Express
- DBMS : MongoDB
- ODM : Mongoose
- API : Dialogflow
- HTML View Engine : Pug
